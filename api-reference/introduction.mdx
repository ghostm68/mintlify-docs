## Overview

The inkrealm Creative API provides programmatic access to creative projects, downloadable content, and system status. All endpoints return JSON and require no authentication.

### Base URL

https://inkrealm.info/api/v1

## Quick Start

```javascript
// Fetch all creative projects
fetch('https://inkrealm.info/api/v1/projects.json')
  .then(res => res.json())
  .then(projects => {
    console.log('Available projects:', projects);
  });

Available Endpoints
Creative Projects
GET /projects.json - List all creative works

GET /projects/{id} - Get specific project (coming soon)

Digital Downloads
GET /downloads.json - List all downloadable files

GET /downloads/{id} - Get specific download info (coming soon)

System Status
GET /status.json - Current system status and uptime
